# OpenSKS Schemas

Run `cargo run -p xtask -- schemas` after changing `opensks-contracts`.

Current generated contracts include engine request/event envelopes, execution
events, work items, model/provider routing, scheduler concurrency, pipeline
graphs, compiled plans, Git isolation, patch envelopes, completion proofs,
hooks, CodeGraph records/indexes, TriWiki records, context packs, image
assets/ledgers, reasoning reports, outbox items, the tracked/local data-plane
manifest, retention plans, and release proofs.
